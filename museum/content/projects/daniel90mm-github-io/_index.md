A static engineering log. Project summaries on the front page; each project has its own page with a sidebar log of dated progress entries. KaTeX for math, TikZ via TikZJax for diagrams, a `fig` shortcode for raster + inline-SVG figures.

The hero rotates through ~250 pyfiglet/Calligraphy ASCII fonts; you can blocklist any font you don't like and it never appears again in your browser.
