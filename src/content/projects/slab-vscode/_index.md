status: live

The Slab editor is great, but for course exercises that live as `.ipynb` files in a VS Code workspace, building inside the platform people already use is the pragmatic move. This extension adds:

- Create `.ipynb` notebooks from markdown scaffolds
- Export notebooks back to review-friendly markdown
- A handful of keystroke fixes that make notebook editing feel less hostile
