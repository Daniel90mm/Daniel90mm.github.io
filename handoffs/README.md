# Handoffs

Session handoff files written by the agent when coherence degrades, the user is frustrated, or a natural phase boundary is hit. **Not** triggered by conversation length or proximity to auto-compaction.

Naming: `SESSION_<YYYY-MM-DD_HHMM>.md`

See `CLAUDE.md` → Protocols → Session handoff for contents.

To resume a session: *"Resume from the latest file in `handoffs/`."*
